# BOC 净值两日对比分析

读取两个日期的中行理财净值 CSV，合并共同产品，计算净值差，并按持有期过滤。

## 数据源

- 文件编码：**GBK**（中文 Windows 导出常见编码）
- 产品代码列名：**0422 文件**为 `产品代码`，**0423 文件**为 `产品编码`，需统一处理
- 关键字段：`产品名称`、`最新单位净值`、`净值更新日期`、`个人认购总额度（元）`、`个人剩余额度（元）`、`产品规模（元）`

## 工作流程

### Step 1：读取两个 CSV

使用 `scripts/compare_nav.py` 执行完整处理流程，或手动按以下步骤执行：

```python
import csv

def read_csv(path, code_col):
    data = {}
    with open(path, encoding='gbk') as f:
        reader = csv.DictReader(f)
        for row in reader:
            code = row[code_col].strip()
            data[code] = row
    return data

data_0422 = read_csv('<0422文件路径>', '产品代码')
data_0423 = read_csv('<0423文件路径>', '产品编码')
common_codes = set(data_0422.keys()) & set(data_0423.keys())
```

### Step 2：持有期过滤规则（`should_remove` 函数）

按产品名称匹配持有期关键词，**剔除**以下产品：

| 类型 | 规则 | 保留/剔除 |
|------|------|----------|
| 天（阿拉伯） | `\d+天` > 30天 | 剔除 |
| 天（中文） | `零一二三四五六七八九十百+天` > 30天 | 剔除 |
| 月（阿拉伯） | `\d+个月` ≥ 2 | 剔除 |
| 月（中文） | `一二三四五六七八九十+个月`，排除 `一`/`两`/`零` | 剔除 |
| 年 | 任意 `X年`（阿拉伯或中文） | 剔除 |
| 周期关键词 | 含 `季季开`、`季添利`、`年年开`、`半年开`、`半年购`、`双周开`、`六个月` | 剔除 |

**保留**：`1个月`、`一个月` 等1个月产品。

**中文数字解析**：`零/一/二/三/四/五/六/七/八/九/十`，支持 `十`（=10）、`十几`（如 `十五`=15）、`几十`（如 `二十`=20）。

### Step 3：计算净值差

```python
nav_diff = (float(nav_0423) - float(nav_0422)) * 10000
diff_str = f'{nav_diff:.2f}'  # 保留两位小数
```

### Step 4：金额转万元

```python
def to_wan(val):
    try:
        return f'{float(val) / 10000:.2f}'
    except:
        return val
```

### Step 5：输出字段

| 字段 | 说明 |
|------|------|
| 产品名称 | — |
| 产品代码 | — |
| 产品规模（万元） | 转万元 |
| 最新单位净值(新日期) | — |
| 最新单位净值(旧日期) | — |
| 净值差(×10000) | (新-旧)×10000，2位小数 |
| 个人认购总额度（万元） | 转万元 |
| 个人剩余额度（万元） | 转万元 |
